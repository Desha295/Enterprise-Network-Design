# Enterprise Multi-Site Network Design

Enterprise-level network designed using Cisco Packet Tracer.

## Features

- EIGRP Routing
- HSRP Redundancy
- VLAN Segmentation
- ACL Security Policies
- DHCP Services
- PAT Configuration
- EtherChannel
- STP Root and Secondary Switches
- SSH Hardening
- Site-to-Site VPN

## Technologies

- Cisco Packet Tracer

- Routing & Switching
- Network Security
